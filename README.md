# Chainlink Style Price Feed Using Tellor
It's permissionless.

## Pre-requisites
[foundry](https://github.com/foundry-rs/foundry)

## How to use
Check out `/test/ETHUSDPriceFeed.t.sol` for an example of how to use the price feed.

## Run tests
```
forge test
```

## Fmt code

forge fmt
